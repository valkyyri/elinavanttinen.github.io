Koodaus

Toimin Demo lab -opintojaksolla päätoimisesti koodarina. Tällä sivulla kerron projektissa tekemästäni koodista ja omasta kehityksestäni opintojakson aikana.

Oppimistavoitteeni

Tavoitteeni tälle opintojaksolle oli syventää Angularin osaamista, sillä Angularin perusteetkin tuntuivat vielä projektin alussa epäselviltä. Toinen tavoite oli oppia käyttämään Gittiä ryhmätyössä. Olin käyttänyt sitä aiemminkin, mutta yhteistyö usean koodarin kanssa hieman pelotti aiemmin koettujen merge-ongelmien vuoksi. Ensin tavoitteet ja niiden saavuttaminen: Angular, Git, API:n käyttö. Esimerkkejä koodista. Bugien korjaamista.

Vertailufunktio samassa tiedostossa:

                                        
                                            compare(ingredients) {
                                                this.alServ.getAllergens().subscribe((allergens) => {
                                                  let stringredients = JSON.stringify(ingredients).toLowerCase();
                                                  allergens.map((allergen) => {
                                                    let response = stringredients.match(allergen.name.toLowerCase());
                                                    //check if a set allergen was found in ingredients
                                                    if (response !== null && allergen.status === true) {
                                                      let isAlreadyFound: boolean;
                                                      isAlreadyFound = this.foundAl.includes(response[0]);
                                                      //avoid repeating the same allergen
                                                      if (isAlreadyFound === false) {
                                                        this.foundAl.push(response[0]);
                                                      }
                                                    }
                                                    /* String.match returns null if the allergen is not found,
                                                  and an array (with the allergen as first item) if the allergen is found */
                                                  });
                                                });
                                              }
